## Listar Puntos GPS

<RequestExample>
```bash cURL

  -H "Authorization: Bearer {token}"
```

```javascript JavaScript

  headers: {
    'Authorization': 'Bearer ' + token

});
```


## Registrar Punto GPS

<api>POST /api/gps-points</api>

<RequestExample>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/gps-points \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "vehiculo_id": 1,
    "latitud": -12.0464,
    "longitud": -77.0428,
    "velocidad": 45.5,
    "direccion": "NE",
    "captured_at": "2024-03-15T14:30:00"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/gps-points', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    vehiculo_id: 1,
    latitud: -12.0464,
    longitud: -77.0428,
    velocidad: 45.5,
    direccion: 'NE',
    captured_at: '2024-03-15T14:30:00'
  })
});

const gpsPoint = await response.json();
```

```php PHP
$gpsPoint = Http::withToken($token)
    ->post('https://api.fabricamarie.com/api/gps-points', [
        'vehiculo_id' => 1,
        'latitud' => -12.0464,
        'longitud' => -77.0428,
        'velocidad' => 45.5,
        'direccion' => 'NE',
        'captured_at' => '2024-03-15T14:30:00'
    ])
    ->json();
```
</RequestExample>

<ParamField body="vehiculo_id" type="integer" required>
  ID del vehículo para el cual se registra el punto GPS
</ParamField>

<ParamField body="latitud" type="number" required>
  Latitud del punto GPS (formato decimal, ej: -12.0464)
</ParamField>

<ParamField body="longitud" type="number" required>
  Longitud del punto GPS (formato decimal, ej: -77.0428)
</ParamField>

<ParamField body="velocidad" type="number">
  Velocidad en km/h al momento de la captura
</ParamField>

<ParamField body="direccion" type="string">
  Dirección cardinal: N, S, E, W, NE, NW, SE, SW
</ParamField>

<ParamField body="captured_at" type="string" required>
  Fecha y hora de captura del punto GPS (formato ISO 8601)
</ParamField>

<ResponseExample>
```json Respuesta Exitosa

  "id": 101,
  "vehiculo_id": 1,
  "latitud": -12.0464,
  "longitud": -77.0428,

  "captured_at": "2024-03-15T14:30:00.000000Z",
  "created_at": "2024-03-15T14:30:05.000000Z",
```

## Actualizar Punto GPS

<RequestExample>
```bash cURL

  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "velocidad": 50.0,
    "direccion": "N"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/gps-points/100', {
  method: 'PUT',

    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'

    velocidad: 50.0,
```

## Casos de Uso

El sistema de rastreo GPS permite:

1. **Monitoreo en Tiempo Real**: Seguimiento de la ubicación actual de cada vehículo
2. **Historial de Rutas**: Consulta de trayectorias pasadas para análisis
3. **Control de Velocidad**: Registro de velocidades para seguridad vial
4. **Geolocalización de Ventas**: Correlación de ubicaciones con actividades de venta
5. **Optimización de Rutas**: Análisis de patrones de desplazamiento

<Tip>
Para una mejor precisión, se recomienda enviar puntos GPS cada 30-60 segundos cuando el vehículo está en movimiento.
</Tip>
