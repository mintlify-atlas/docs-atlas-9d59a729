## Listar Vehículos Activos

<RequestExample>
```bash cURL
curl https://api.fabricamarie.com/api/vehiculos \
  -H "Authorization: Bearer {token}"
```

```javascript JavaScript

  headers: {
    'Authorization': 'Bearer ' + token
  }
});

const vehiculos = await response.json();
```

```php PHP
$vehiculos = Http::withToken($token)
```

## Crear Vehículo

<api>POST /api/vehiculos</api>

<RequestExample>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/vehiculos \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "placa": "ABC-123",
    "marca": "Toyota",
    "modelo": "Hilux",
    "año": 2022,
    "color": "Blanco",
    "estado": "DISPONIBLE",
    "kilometraje": 0
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/vehiculos', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    placa: 'ABC-123',
    marca: 'Toyota',
    modelo: 'Hilux',
    año: 2022,
    color: 'Blanco',
    estado: 'DISPONIBLE',
    kilometraje: 0
  })
});

const vehiculo = await response.json();
```

```php PHP
$vehiculo = Http::withToken($token)
    ->post('https://api.fabricamarie.com/api/vehiculos', [
        'placa' => 'ABC-123',
        'marca' => 'Toyota',
        'modelo' => 'Hilux',
        'año' => 2022,
        'color' => 'Blanco',
        'estado' => 'DISPONIBLE',
        'kilometraje' => 0
    ])
    ->json();
```
</RequestExample>

<ParamField body="placa" type="string" required>
  Placa del vehículo
</ParamField>

<ParamField body="marca" type="string" required>
  Marca del vehículo
</ParamField>

<ParamField body="modelo" type="string" required>
  Modelo del vehículo
</ParamField>

<ParamField body="año" type="integer" required>
  Año del vehículo
</ParamField>

<ParamField body="color" type="string" required>
  Color del vehículo
</ParamField>

<ParamField body="estado" type="string">
  Estado del vehículo: DISPONIBLE, EN_USO, o MANTENIMIENTO (por defecto: DISPONIBLE)
</ParamField>

<ParamField body="kilometraje" type="integer">
  Kilometraje actual del vehículo
</ParamField>

<ResponseExample>
```json Respuesta Exitosa
{
  "id": 3,
  "placa": "ABC-123",
  "marca": "Toyota",
  "modelo": "Hilux",
  "año": 2022,
  "color": "Blanco",
  "estado": "DISPONIBLE",
  "activo": true,
  "kilometraje": 0,
  "created_at": "2024-03-15T14:30:00.000000Z",
  "updated_at": "2024-03-15T14:30:00.000000Z"
}
```
</ResponseExample>

## Actualizar Vehículo

<RequestExample>
```bash cURL

  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "estado": "EN_USO",
    "kilometraje": 45500
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/vehiculos/1', {
  method: 'PUT',
```

## Asignar Vehículo a Vendedor

<api>POST /api/vehiculos/{id}/asignar</api>

<RequestExample>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/vehiculos/1/asignar \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "vendedor_id": 5
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/vehiculos/1/asignar', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    vendedor_id: 5
  })
});

const result = await response.json();
```
</RequestExample>

<ParamField path="id" type="integer" required>
  ID del vehículo a asignar
</ParamField>

<ParamField body="vendedor_id" type="integer" required>
  ID del vendedor al que se asignará el vehículo
</ParamField>

<ResponseExample>
```json Respuesta Exitosa
{
  "id": 1,
  "placa": "ABC-123",
  "marca": "Toyota",
  "modelo": "Hilux",
  "estado": "EN_USO"
}
```

```json Error: Ya Asignado
{
  "message": "El vehiculo ya se encuentra asignado al vendedor"
}
```
</ResponseExample>

## Desactivar Vehículo

<api>DELETE /api/vehiculos/{id}</api>

<RequestExample>
```bash cURL
curl -X DELETE https://api.fabricamarie.com/api/vehiculos/1 \
  -H "Authorization: Bearer {token}"
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/vehiculos/1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer ' + token
  }
});

const result = await response.json();
```
</RequestExample>

<ParamField path="id" type="integer" required>
  ID del vehículo a desactivar
</ParamField>

<ResponseExample>
```json Respuesta Exitosa
{
  "message": "Vehículo desactivado"
}
```
</ResponseExample>

<Note>
Este endpoint realiza una eliminación suave (soft delete), marcando el vehículo como inactivo en lugar de eliminarlo permanentemente de la base de datos.
</Note>
