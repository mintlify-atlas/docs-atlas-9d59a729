## Descripción

Permite autenticar un usuario del sistema mediante email y contraseña. Al autenticarse exitosamente, retorna un token de acceso Sanctum que debe utilizarse en las demás peticiones API.

<Note>
Este endpoint **no requiere autenticación** previa.
</Note>

## Parámetros de la solicitud

<ParamField body="email" type="string" required>
  Correo electrónico del usuario registrado en el sistema.
  
  **Validación:** Debe ser un email válido.
</ParamField>

<ParamField body="password" type="string" required>
  Contraseña del usuario.
  
  **Validación:** Campo requerido.
</ParamField>

## Respuesta exitosa

<ResponseField name="token" type="string">
  Token de acceso personal (Sanctum) para autenticar futuras peticiones.
</ResponseField>

  Información completa del usuario autenticado.

## Ejemplos de código

<CodeGroup>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/login \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "email": "admin@fabricamarie.com",
    "password": "password123"
  }'
```

```javascript JavaScript (Fetch)
const response = await fetch('https://api.fabricamarie.com/api/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Accept': 'application/json'
  },
  body: JSON.stringify({
    email: 'admin@fabricamarie.com',
    password: 'password123'
  })
});

const data = await response.json();

if (response.ok) {
  // Guardar el token para futuras peticiones
  localStorage.setItem('auth_token', data.token);
  console.log('Usuario autenticado:', data.user);
} else {
  console.error('Error de autenticación:', data.message);
}
```

```php PHP
<?php

use Illuminate\Support\Facades\Http;

$response = Http::post('https://api.fabricamarie.com/api/login', [
    'email' => 'admin@fabricamarie.com',
    'password' => 'password123'
]);

if ($response->successful()) {
    $data = $response->json();
    $token = $data['token'];
    $user = $data['user'];
    $roles = $data['roles'];
    $permisos = $data['permisos'];
    
    // Usar el token en futuras peticiones
    echo "Token: " . $token;
} else {
    echo "Error: " . $response->json()['message'];
}
```

```python Python
import requests

url = 'https://api.fabricamarie.com/api/login'
headers = {
    'Content-Type': 'application/json',
    'Accept': 'application/json'
}
data = {
    'email': 'admin@fabricamarie.com',
    'password': 'password123'
}

response = requests.post(url, json=data, headers=headers)

if response.status_code == 200:
    result = response.json()
    token = result['token']
    user = result['user']
    print(f"Token: {token}")
    print(f"Usuario: {user['nombre']}")
else:
    print(f"Error: {response.json()['message']}")
```

## Notas adicionales

<Info>
- El usuario debe tener el campo `activo` en `1` para poder autenticarse
- El campo `ultimo_login` se actualiza automáticamente en cada inicio de sesión exitoso
- El token generado es de tipo **Sanctum Personal Access Token** con el nombre `erp-token`
- Guarda el token de forma segura para utilizarlo en el header `Authorization: Bearer {token}` en futuras peticiones
</Info>

<Warning>
No expongas el token en URLs ni lo almacenes en lugares inseguros. Utiliza HTTPS para todas las comunicaciones.
</Warning>
