## Crear Movimiento de Caja

<RequestExample>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/caja/movimiento \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "tipo": "INGRESO",
    "monto": 500.00,
    "categoria": "VENTA",
    "descripcion": "Pago de cliente",
    "referencia_tipo": "PEDIDO",
    "referencia_id": 456
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/caja/movimiento', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    tipo: 'INGRESO',
    monto: 500.00,
    categoria: 'VENTA',
    descripcion: 'Pago de cliente',
    referencia_tipo: 'PEDIDO',
    referencia_id: 456
  })
});

const movimiento = await response.json();
```

```php PHP
$movimiento = Http::withToken($token)
    ->post('https://api.fabricamarie.com/api/caja/movimiento', [
        'tipo' => 'INGRESO',
        'monto' => 500.00,
        'categoria' => 'VENTA',
        'descripcion' => 'Pago de cliente',
        'referencia_tipo' => 'PEDIDO',
        'referencia_id' => 456
    ])
    ->json();
```
</RequestExample>

<ParamField body="tipo" type="string" required>
  Tipo de movimiento: `INGRESO` o `EGRESO`
</ParamField>

<ParamField body="monto" type="number" required>

<ParamField body="categoria" type="string" required>

<ParamField body="descripcion" type="string" required>

  Tipo de movimiento (INGRESO o EGRESO)
</ResponseField>

<ResponseField name="monto" type="number">
  Monto del movimiento
</ResponseField>

<ResponseField name="categoria" type="string">
  Categoría del movimiento
</ResponseField>

<ResponseField name="descripcion" type="string">

<ResponseField name="referencia_tipo" type="string">

## Obtener Todos los Movimientos

```bash cURL

  -H "Authorization: Bearer {token}"
```


## Categorías de Movimientos

Las categorías más comunes de movimientos incluyen:

- **VENTA**: Ingresos por ventas
- **SALIDA**: Egresos por salidas de caja
- **VIATICO**: Egresos por viáticos a vendedores
- **VUELTO**: Ingresos por vueltos de salidas o viáticos liquidados
- **AJUSTE**: Ajustes de caja
- **DEPOSITO**: Depósitos bancarios
- **RETIRO**: Retiros de efectivo
