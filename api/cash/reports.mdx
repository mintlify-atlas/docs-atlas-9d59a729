## Obtener Reporte por ID de Caja

<RequestExample>
```bash cURL
curl https://api.fabricamarie.com/api/caja/123/reporte \
  -H "Authorization: Bearer {token}"
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/caja/123/reporte', {
  headers: {
    'Authorization': 'Bearer ' + token
  }
});

const reporte = await response.json();
```

```php PHP
$reporte = Http::withToken($token)
    ->get('https://api.fabricamarie.com/api/caja/123/reporte')
    ->json();
```
</RequestExample>

<ParamField path="id" type="integer" required>
  ID de la caja para generar el reporte
</ParamField>

<ResponseField name="caja_id" type="integer">
  ID de la caja
</ResponseField>

<ResponseField name="fecha" type="string">
  Fecha de la caja
</ResponseField>

<ResponseField name="usuario" type="object">
  Información del usuario que abrió la caja
</ResponseField>

<ResponseField name="saldo_inicial" type="number">
  Saldo inicial de la caja
</ResponseField>

<ResponseField name="saldo_actual" type="number">
  Saldo actual de la caja
</ResponseField>

<ResponseField name="total_ingresos" type="number">
  Total de ingresos del día
</ResponseField>

<ResponseField name="total_egresos" type="number">
  Total de egresos del día
</ResponseField>

<ResponseField name="movimientos" type="array">
  Array con todos los movimientos de la caja
</ResponseField>

<ResponseField name="estado" type="string">
  Estado de la caja (ABIERTA o CERRADA)
</ResponseField>

## Obtener Reporte por Fecha

<api>POST /api/caja/reporte/fecha</api>

<RequestExample>
```bash cURL
curl -X POST https://api.fabricamarie.com/api/caja/reporte/fecha \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "fecha": "2024-03-15"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fabricamarie.com/api/caja/reporte/fecha', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    fecha: '2024-03-15'
  })
});

const reporte = await response.json();
```

```php PHP
$reporte = Http::withToken($token)
    ->post('https://api.fabricamarie.com/api/caja/reporte/fecha', [
        'fecha' => '2024-03-15'
    ])
    ->json();
```
</RequestExample>

## Resumen del Reporte

El reporte de caja incluye:

1. **Información General**: Fecha, usuario, estado de la caja
2. **Saldos**: Inicial, actual y diferencia
3. **Totales**: Suma de ingresos y egresos
4. **Movimientos Detallados**: Lista completa de transacciones
5. **Categorización**: Movimientos agrupados por tipo y categoría

Este reporte es útil para:
- Auditorías diarias
- Reconciliación de efectivo
- Análisis de flujo de caja
- Detección de diferencias o faltantes
