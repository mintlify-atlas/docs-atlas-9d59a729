## Reporte de Ventas con Resumen

<Note>
  Genera un reporte de ventas con filtros opcionales y un resumen estadístico.
</Note>

### Endpoint

```
GET /api/ventas/reporte
```

### Autenticación

<ParamField header="Authorization" type="string" required>
  Token de autenticación Bearer
</ParamField>

### Parámetros de Consulta (Filtros)

<ParamField query="fecha_desde" type="string">
  Fecha de inicio (formato: Y-m-d)
</ParamField>

<ParamField query="fecha_hasta" type="string">
  Fecha de fin (formato: Y-m-d)
</ParamField>

<ParamField query="cliente_id" type="integer">
  Filtrar por cliente
</ParamField>

<ParamField query="vendedor_id" type="integer">
  Filtrar por vendedor
</ParamField>

<ParamField query="producto_id" type="integer">
  Filtrar por producto
</ParamField>

<ParamField query="tipo_pago" type="string">
  Filtrar por tipo de pago (CONTADO/CREDITO)
</ParamField>

### Respuesta

<ResponseField name="filtros" type="object">
  Objeto con los filtros aplicados al reporte
  - `fecha_desde`: Fecha inicial aplicada
  - `fecha_hasta`: Fecha final aplicada
  - `cliente_id`: ID de cliente filtrado
  - `vendedor_id`: ID de vendedor filtrado
  - `producto_id`: ID de producto filtrado
  - `tipo_pago`: Tipo de pago filtrado
</ResponseField>

<ResponseField name="resumen" type="object">
  Resumen estadístico del reporte
  
  <Expandable title="Propiedades">
    <ResponseField name="cantidad_ventas" type="integer">
      Total de ventas en el periodo
    </ResponseField>
    
    <ResponseField name="total_vendido" type="number">
      Suma total de ventas (contado + crédito)
    </ResponseField>
    
    <ResponseField name="total_contado" type="number">
      Suma total de ventas de contado
    </ResponseField>
    
    <ResponseField name="total_credito" type="number">
      Suma total de ventas a crédito
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="ventas" type="array">
  Lista detallada de ventas con las mismas propiedades que el endpoint de listado
</ResponseField>

<CodeGroup>

```bash Solicitud sin filtros
curl -X GET "https://api.fabricamarie.com/api/ventas/reporte" \
  -H "Authorization: Bearer {token}"
```

```bash Solicitud con filtros
curl -X GET "https://api.fabricamarie.com/api/ventas/reporte?fecha_desde=2026-03-01&fecha_hasta=2026-03-11&vendedor_id=12" \
  -H "Authorization: Bearer {token}"
```

## Exportar Reporte a Excel

<Note>
  Descarga un reporte de ventas en formato Excel (.xls) con los mismos filtros del reporte.
</Note>

### Endpoint

```
GET /api/ventas/exportar-excel
```

### Autenticación

<ParamField header="Authorization" type="string" required>
  Token de autenticación Bearer
</ParamField>

### Parámetros de Consulta (Filtros)

Acepta los mismos filtros que el endpoint de reporte:

<ParamField query="fecha_desde" type="string">
  Fecha de inicio (formato: Y-m-d)
</ParamField>

<ParamField query="fecha_hasta" type="string">
  Fecha de fin (formato: Y-m-d)
</ParamField>

<ParamField query="cliente_id" type="integer">
  Filtrar por cliente
</ParamField>

<ParamField query="vendedor_id" type="integer">
  Filtrar por vendedor
</ParamField>

<ParamField query="producto_id" type="integer">
  Filtrar por producto
</ParamField>

<ParamField query="tipo_pago" type="string">
  Filtrar por tipo de pago (CONTADO/CREDITO)
</ParamField>

### Respuesta

Retorna un archivo Excel descargable con:
- Nombre de archivo: `reporte_ventas_YYYYMMDD_HHMMSS.xls`
- Content-Type: `application/vnd.ms-excel; charset=UTF-8`
- Codificación UTF-8 con BOM para correcta visualización de caracteres especiales

### Contenido del Excel

**Columnas:**
1. Código
2. Fecha
3. Cliente
4. Vendedor
5. Productos (lista separada por comas)
6. Tipo de pago
7. Total neto

**Resumen al final:**
- Total vendido
- Total contado
- Total crédito

<CodeGroup>

```bash Descargar reporte completo
curl -X GET "https://api.fabricamarie.com/api/ventas/exportar-excel" \
  -H "Authorization: Bearer {token}" \
  -o reporte_ventas.xls
```

```bash Descargar con filtros
curl -X GET "https://api.fabricamarie.com/api/ventas/exportar-excel?fecha_desde=2026-03-01&fecha_hasta=2026-03-31&tipo_pago=CREDITO" \
  -H "Authorization: Bearer {token}" \
  -o reporte_ventas_credito.xls
```

```javascript JavaScript (Descarga)
const response = await fetch(
  'https://api.fabricamarie.com/api/ventas/exportar-excel?fecha_desde=2026-03-01&fecha_hasta=2026-03-31',
  {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const blob = await response.blob();
const url = window.URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = `reporte_ventas_${new Date().getTime()}.xls`;
document.body.appendChild(a);
a.click();
a.remove();
```

```python Python (Descarga)
import requests
from datetime import datetime

url = "https://api.fabricamarie.com/api/ventas/exportar-excel"
params = {
    "fecha_desde": "2026-03-01",
    "fecha_hasta": "2026-03-31"
}
headers = {
    "Authorization": f"Bearer {token}"
}

response = requests.get(url, params=params, headers=headers)

if response.status_code == 200:
    filename = f"reporte_ventas_{datetime.now().strftime('%Y%m%d_%H%M%S')}.xls"
    with open(filename, 'wb') as f:
        f.write(response.content)
    print(f"Reporte descargado: {filename}")
```

</CodeGroup>

## Formato del Archivo Excel

El archivo generado usa formato XML de Excel 2003 con las siguientes características:

- **Formato**: SpreadsheetML (XML)
- **Extensión**: .xls
- **Codificación**: UTF-8 con BOM
- **Compatible**: Excel 2003 y versiones posteriores, LibreOffice, Google Sheets
- **Estructura**:
  - Hoja única: "Reporte Ventas"
  - Encabezados en la primera fila
  - Datos de ventas ordenados por fecha descendente
  - Fila de totales al final con formato numérico

## Casos de Uso

### Reporte mensual de ventas
```
GET /api/ventas/reporte?fecha_desde=2026-03-01&fecha_hasta=2026-03-31
```

### Reporte de ventas a crédito pendientes
```
GET /api/ventas/reporte?tipo_pago=CREDITO
```

### Desempeño de un vendedor
```
GET /api/ventas/reporte?vendedor_id=12&fecha_desde=2026-03-01&fecha_hasta=2026-03-31
```

### Exportar ventas de un cliente
```
GET /api/ventas/exportar-excel?cliente_id=45
```
