## Descripción General

El sistema de routing de Fabrica Marie ERP utiliza **React Router v6** con rutas protegidas, rutas públicas y un sistema de navegación basado en roles.

## Configuración de React Router

El archivo `App.tsx` contiene toda la configuración de rutas:

<CodeGroup>
```tsx App.tsx (configuración principal)
import { BrowserRouter, Routes, Route, Navigate } from "react-router-dom";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { RoleProvider } from "@/contexts/RoleContext";
import { AuthProvider, useAuth } from "@/contexts/AuthContext";
import { MainLayout } from "@/components/layout/MainLayout";
import Login from "@/components/auth/login";

const queryClient = new QueryClient();

const App = () => (
  <QueryClientProvider client={queryClient}>
    <AuthProvider>
      <RoleProvider>
        <TooltipProvider>
          <Toaster />
          <Sonner />
          <BrowserRouter>
            <AppRoutes />
          </BrowserRouter>
        </TooltipProvider>
      </RoleProvider>
    </AuthProvider>
  </QueryClientProvider>
);

export default App;
```
</CodeGroup>

## Componentes de Protección de Rutas

### ProtectedRoute

Protege rutas que requieren autenticación:

<CodeGroup>
```tsx ProtectedRoute Component
const ProtectedRoute = ({ children }: { children: JSX.Element }) => {
  const { isAuthenticated, loading } = useAuth();

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-blue-600"></div>
      </div>
    );
  }

  if (!isAuthenticated()) {
    return <Navigate to="/login" replace />;
  }

  return children;
};
```
</CodeGroup>

### PublicRoute

Redirige usuarios autenticados al dashboard:

<CodeGroup>
```tsx PublicRoute Component
const PublicRoute = ({ children }: { children: JSX.Element }) => {
  const { isAuthenticated, loading } = useAuth();

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-blue-600"></div>
      </div>
    );
  }

  if (isAuthenticated()) {
    return <Navigate to="/" replace />;
  }

  return children;
};
```
</CodeGroup>

### RootHandler

Maneja la ruta raíz con redirección inteligente:

<CodeGroup>
```tsx RootHandler Component
const RootHandler = () => {
  const { isAuthenticated, loading } = useAuth();

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-blue-600"></div>
      </div>
    );
  }

  // Si está autenticado, va al dashboard
  // Si no, va al login
  return isAuthenticated()
    ? <Navigate to="/dashboard" replace />
    : <Navigate to="/login" replace />;
};
```
</CodeGroup>

## Estructura de Rutas

### Rutas Públicas

<CodeGroup>
```tsx Rutas públicas
<Routes>
  {/* Ruta raíz - redirige según autenticación */}
  <Route path="/" element={<RootHandler />} />

  {/* Login - solo accesible si NO está autenticado */}
  <Route path="/login" element={
    <PublicRoute>
      <Login />
    </PublicRoute>
  } />
</Routes>
```
</CodeGroup>

## Navegación Programática

### useNavigate Hook

<CodeGroup>
```tsx Navegación con useNavigate
import { useNavigate } from 'react-router-dom';

const MyComponent = () => {
  const navigate = useNavigate();

  const handleSave = async () => {
    await saveData();
    // Navegar a otra ruta
    navigate('/ventas/historial');
  };

  const handleCancel = () => {
    // Navegar hacia atrás
    navigate(-1);
  };

  return (
    <div>
      <Button onClick={handleSave}>Guardar</Button>
      <Button onClick={handleCancel}>Cancelar</Button>
    </div>
  );
};
```
</CodeGroup>

### Query Parameters

<CodeGroup>
```tsx Uso de query params
import { useSearchParams } from 'react-router-dom';

const ClientsList = () => {
  const [searchParams, setSearchParams] = useSearchParams();
  
  const filter = searchParams.get('filter') || 'all';
  const search = searchParams.get('search') || '';

  const handleFilterChange = (newFilter: string) => {
    setSearchParams({ filter: newFilter, search });
  };

  return (
    <div>
      <Input 
        value={search}
        onChange={(e) => setSearchParams({ filter, search: e.target.value })}
      />
    </div>
  );
};
```

## Lazy Loading de Rutas

Para optimizar el rendimiento, puedes usar lazy loading:

<CodeGroup>
```tsx Lazy loading
import { lazy, Suspense } from 'react';

const Dashboard = lazy(() => import('@/pages/Dashboard'));
const ClientsList = lazy(() => import('@/pages/clients/ClientsList'));

const App = () => (
  <Suspense fallback={<LoadingSpinner />}>
    <Routes>
      <Route path="/dashboard" element={<Dashboard />} />
      <Route path="/clientes" element={<ClientsList />} />
    </Routes>
  </Suspense>
);
```
</CodeGroup>

## Control de Acceso por Roles

Las rutas se pueden proteger según el rol del usuario:

<CodeGroup>
```tsx Protección por rol
import { useRole } from '@/contexts/RoleContext';
import { Navigate } from 'react-router-dom';

const AdminRoute = ({ children }: { children: JSX.Element }) => {
  const { currentRole } = useRole();

  if (currentRole !== 'ADMIN' && currentRole !== 'GERENTE') {
    return <Navigate to="/dashboard" replace />;
  }

  return children;
};

// Uso
<Route path="/configuracion" element={
  <AdminRoute>
    <SettingsPage />
  </AdminRoute>
} />
```
</CodeGroup>

## Navegación en el Sidebar

El componente `Sidebar` mapea las rutas a elementos de navegación:

<CodeGroup>
```tsx Sidebar navigation
import { useLocation, Link } from 'react-router-dom';
import { Home, Package, ShoppingCart, Users } from 'lucide-react';

const Sidebar = () => {
  const location = useLocation();

  const navItems = [
    { path: '/dashboard', icon: Home, label: 'Dashboard' },
    { path: '/almacen', icon: Package, label: 'Almacén' },
    { path: '/ventas', icon: ShoppingCart, label: 'Ventas' },
    { path: '/clientes', icon: Users, label: 'Clientes' },
  ];

  return (
    <nav>
      {navItems.map((item) => (
        <Link
          key={item.path}
          to={item.path}
          className={location.pathname.startsWith(item.path) ? 'active' : ''}
        >
          <item.icon />
          {item.label}
        </Link>
      ))}
    </nav>
  );
};
```
</CodeGroup>
