## Descripción General

Fabrica Marie ERP ofrece un completo sistema de reportes para análisis de negocio, toma de decisiones y auditoría. Esta guía describe todos los reportes disponibles y cómo utilizarlos.

<Info>
  **Acceso a reportes:** Varía según rol. GERENTE y ADMIN tienen acceso completo.
</Info>

## Reportes de Ventas

### Reporte General de Ventas

El reporte más completo, con múltiples filtros:

```bash
GET /api/ventas/reporte
Authorization: Bearer {token}
```

**Parámetros disponibles:**

<Accordion title="Filtros de Fecha">
- `fecha_desde`: Fecha inicial (YYYY-MM-DD)
- `fecha_hasta`: Fecha final (YYYY-MM-DD)

**Ejemplo:**
```bash
GET /api/ventas/reporte?fecha_desde=2026-03-01&fecha_hasta=2026-03-11
```
</Accordion>

<Accordion title="Filtros de Cliente y Vendedor">
- `cliente_id`: Filtrar por un cliente específico
- `vendedor_id`: Filtrar por un vendedor específico

**Ejemplo:**
```bash
GET /api/ventas/reporte?vendedor_id=5&fecha_desde=2026-03-01
```
</Accordion>

<Accordion title="Filtros de Producto y Tipo de Pago">
- `producto_id`: Ventas que incluyan un producto específico
- `tipo_pago`: CONTADO o CREDITO

**Ejemplo:**
```bash
GET /api/ventas/reporte?tipo_pago=CREDITO&fecha_desde=2026-03-01
```
</Accordion>

### Exportar Ventas a Excel

Descarga el reporte en formato Excel:

```bash
GET /api/ventas/exportar-excel?fecha_desde=2026-03-01&fecha_hasta=2026-03-11
Authorization: Bearer {token}
```

Esto descarga un archivo `.xls` con:
- Listado completo de ventas
- Código de venta
- Fecha
- Cliente
- Vendedor
- Productos vendidos
- Tipo de pago
- Total neto
- **Totales consolidados**:
  - Total vendido
  - Total contado
  - Total crédito

### Ejemplo de Exportación

```javascript
// Exportar ventas a Excel
async function exportarVentas() {
  const params = new URLSearchParams({
    fecha_desde: '2026-03-01',
    fecha_hasta: '2026-03-11',
    vendedor_id: 5
  });

  const response = await fetch(
    `/api/ventas/exportar-excel?${params}`,
    {
      headers: {
        'Authorization': `Bearer ${token}`
      }
    }
  );

  const blob = await response.blob();
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `ventas_${Date.now()}.xls`;
  a.click();
}
```

## Código de Referencia

**VentaController.php** (`app/Http/Controllers/Api/VentaController.php:29-52`)

```php
public function reporte(Request $request)
{
    $ventas = $this->buildReporteQuery($request)->get();

    $resumen = [
        'cantidad_ventas' => $ventas->count(),
        'total_vendido' => (float) $ventas->sum('total_neto'),
        'total_contado' => (float) $ventas->where('tipo_pago', 'CONTADO')->sum('total_neto'),
        'total_credito' => (float) $ventas->where('tipo_pago', '!=', 'CONTADO')->sum('total_neto'),
    ];

    return response()->json([
        'filtros' => $request->only([...]),
        'resumen' => $resumen,
        'ventas' => $ventas,
    ]);
}
```
